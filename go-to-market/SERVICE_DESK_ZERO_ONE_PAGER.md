# Service Desk Zero - TierZero Pilot Offer

## Big promise
**Start each morning with a smaller queue.**

TierZero handles repetitive service desk work overnight so your human team can focus on exceptions, escalations, and real problems instead of drowning in routine tickets.

## What it does
For one approved workflow, TierZero can:
- read incoming tickets
- pull runbook / KB context
- decide whether to automate, draft, escalate, or ask for missing info
- complete the task or route it cleanly
- leave a full audit trail on the ticket

## Best first workflows
Pick exactly one for the pilot:
1. Password reset / account unlock
2. Access request triage
3. Known software install / fulfillment request
4. Runbook-backed repetitive incidents
